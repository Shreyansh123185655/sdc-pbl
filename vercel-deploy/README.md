# Student PDF Generator - Frontend

Interactive web interface for the Student PDF Generator with Universal Excel Support.

## 🚀 Features

- **Universal Excel Converter** - Accept any Excel file format
- **Smart Structure Detection** - Automatic analysis and conversion
- **Interactive UI** - Drag & drop, progress indicators, animations
- **Real-time Feedback** - File analysis with confidence scoring
- **Responsive Design** - Works on all devices

## 🌐 Deployment

This folder is configured for Vercel deployment:

```bash
# Deploy to Vercel
npx vercel --prod
```

## 🔧 Configuration

- **Framework**: Static HTML/CSS/JavaScript
- **Build Command**: None (static files)
- **Output Directory**: Root
- **Node Version**: Any (static site)

## 📱 Usage

1. **Upload any Excel file** - Drag & drop or browse
2. **Automatic Analysis** - File structure detection
3. **Convert if Needed** - One-click format conversion
4. **Generate PDFs** - Individual student papers
5. **Download Results** - Individual or bulk downloads

## 🎯 Interactive Features

- **Drag & Drop** with visual feedback
- **Progress Indicators** during processing
- **Ripple Effects** on buttons
- **File Removal** functionality
- **Real-time Analysis** display
- **Smooth Animations** and transitions
